<a name="english"></a>
## :gb: English

### Overview

A modern gym landing page built with TypeScript, Vite, and Tailwind CSS. Features responsive design and professional aesthetics for fitness centers.

### Features

- **Responsive Design** - Mobile-first approach
- **Modern UI** - Clean fitness-oriented aesthetics
- **Fast Performance** - Vite-powered build
- **Class Schedule** - Showcase gym classes
- **Membership Plans** - Display pricing options

### Quick Start

```bash
git clone https://github.com/fracabu/gym-landing.git
cd gym-landing
npm install
npm run dev
```

Access at `http://localhost:5173`
